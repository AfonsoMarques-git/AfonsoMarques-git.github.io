Ideia da PAP:
Web site para stand de carros:
Menu: Home / Viaturas ( Viaturas Usadas, Viaturas Novas, Vende a tua viatura) / Contatos / Favoritos / Login e Registro

- Página principal:
	- Filtragem geral
	- Viatura mais recente
	- Últimas vendas
	- Viaturas em destaque
	- Informação de como comprar ou vender

- Viaturas: (abas)
- Viaturas Usadas
		- Filtragem (preço, marca, modelo, combustível, ano do carro, quilómetros)
		- Marcas Disponíveis
		- Carros disponíveis para venda
- Viaturas Novas
		- Filtragem (preço, marca, modelo, combustível, ano do carro, quilómetros)
		- Marcas Disponíveis
		- Carros disponíveis para venda
	- Vende a tua viatura
		- Formulário para caracterização do carro c/fotos para venda
		- Como funciona o processo de venda

- Contatos:
	- Redes Sociais
	- Email
	- Contacto telefónico
	- Formulário de contacto para dúvidas

- Favoritos: (somente com login)
	- Viaturas favoritas
	- Pesquisas guardadas
	- Pop-Up (caso não tenha login)

- Login e Registro: (username e passwd)
	- Página de Login
	- Página de Registro
	- Página de recuperação de password
